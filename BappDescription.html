Integrate Burp Suite with AI Clients using the Model Context Protocol (MCP).

For more information about the protocol visit: modelcontextprotocol.io

Note: Any data sent to external tools via this extension is subject to their data processing policies.

Features

  • Connect Burp Suite to AI clients through MCP
  • Automatic installation for Claude Desktop
  • Comes with packaged Stdio MCP proxy server

Usage

  1. Configure your Burp MCP server in the extension-provided tab.
  2. Configure your MCP client to use the Burp SSE MCP server or stdio proxy.
  3. Interact with Burp through your client.
